# SkillForge AI

SkillForge AI is a personalized learning platform that helps users discover relevant courses, track learning history, and get YouTube-based learning suggestions based on their interests.

---

## Features
- User Login & Signup system  
- Personalized course recommendations  
- Course search functionality  
- Learning history tracking  
- YouTube video suggestions for topics  

---

## Tech Stack
- Python  
- Streamlit  
- Pandas  
- SQLite  

---

## 🔗 Live Demo
https://skillforge-ai-dbxpaapffocu3r8jjwry3a.streamlit.app/

---

## How it Works
- User logs in or signs up  
- Enters learning interests or searches topics  
- System recommends relevant courses/videos  
- Tracks user learning history locally
