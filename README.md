# SkillForge AI

SkillForge AI is a personalized learning recommendation platform built using Streamlit and FastAPI.

## Features
- User Login & Signup
- Personalized Course Recommendations
- Search Courses
- Learning History
- YouTube Video Suggestions

## Tech Stack
- Python
- Streamlit
- FastAPI
- Pandas
- SQLite
